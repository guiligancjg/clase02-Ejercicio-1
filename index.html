Ejercicio 1

Tengo desarrollada la siguiente función en javascript function suma (a,b,callback){ let c = a +b; callback(); }

¿Que es el tercer parámetro recibido?

Respuesta: es una función de callback que se pasa como argumento a otra función y se ejecuta después de que se complete una operación.

¿En que casos es obligatorio desarrollar este tipo de funciones?

Respuesta: Las funciones de callback son útiles en situaciones donde puedas realizar acciones después de que se complete una operación, especialmente en operaciones asincrónicas, en API, manejo de eventos, procesamiento de datos, promesas, temporizadores, validaciones.
